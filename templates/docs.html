Quick Start - Hej Framework

Create a file called app.py and add this code:

import hej

@get('/')
def home():
    return html.html(
        html.head(
            html.title('My App')
        ),
        html.body(
            html.h1('Hello, World!'),
            html.p('Welcome to Hej Framework')
        )
    )

if __name__ == '__main__':
    hej.run()

Run it with:

python app.py

Visit http://127.0.0.1:5000 to see your app!
